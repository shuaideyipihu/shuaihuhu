# 量化金融课程总纲 v1

## 任务书
- 课程主题：从零开始、面向个人投资/交易者的量化金融学习体系
- 学员画像：数学基础中等（学过微积分、线性代数、概率统计，但不精通），会一点 Python，当前属于量化金融初学者
- 最终目标：逐步达到能够独立做基础量化研究、构建和评估策略、使用量化工具进行小规模实操，并以长期可持续盈利为方向
- 教学原则：循序渐进，重直觉、重实操、重风险意识，不承诺盈利结果
- 主线市场：先从股票量化入门，之后扩展到 ETF / 指数，再到期货、期权、加密

---

## 课程目标

### 总目标
建立一套从“金融与数理基础 → 数据与研究 → 策略与回测 → 风险与执行 → 小规模实操”逐步推进的量化学习路径。

### 分阶段目标
1. **理解层**
   - 理解量化金融的基本语言、核心问题与研究方法
   - 建立收益率、波动率、回撤、相关性、因子、回测等基本概念
2. **研究层**
   - 能使用 Python 处理市场数据、做基础统计分析与可视化
   - 能独立完成简单策略的设计、测试和解释
3. **系统层**
   - 能搭建基础研究框架、回测框架和策略评估框架
   - 能识别未来函数、过拟合、幸存者偏差、滑点、容量限制等常见陷阱
4. **实操层**
   - 能对简单策略进行小规模试运行
   - 能建立复盘、风控、记录和迭代机制

### 不承诺的内容
- 不承诺短期盈利
- 不承诺靠单一课程快速形成稳定 alpha
- 不把量化包装成“自动赚钱机器”

## 模块结构

## 模块 0：学习准备与工具环境
目标：搭好整个课程与实操的基础环境。
- Python、Jupyter、pandas、numpy、matplotlib 基础
- 数据文件、notebook、脚本、结果图表的组织方式
- 研究日志与学习记录方法
- 量化学习中的“可复现”思维

## 模块 1：市场语言与量化直觉
目标：学会像量化研究者一样描述市场。
- 股票、ETF、指数、行业、成交量、换手率、做多做空
- 价格、简单收益率、对数收益率、复利
- 回撤、最大回撤
- 为什么量化更关心收益率而不是价格

## 模块 2：风险与统计基础
目标：进入量化分析的数理核心。
- 均值、方差、标准差、波动率
- 协方差、相关性
- 概率分布与抽样直觉
- 假设检验基础
- 滚动窗口分析

## 模块 3：时间序列与市场数据
目标：理解金融数据的“脏、噪、变”。
- 时间序列基本概念
- 平稳性直觉、自相关、趋势与均值回归
- 收益率序列与价格序列的区别
- 数据清洗、缺失值、异常值、复权、幸存者偏差

## 模块 4：单资产策略入门
目标：从概念走向最基础的策略原型。
- 均线与趋势跟踪
- 动量策略基础
- 均值回归基础
- 信号、持仓、收益的关系
- 规则化交易思想

## 模块 5：回测与策略评估
目标：学会检验策略，而不是被回测骗。
- 回测框架基本结构
- 基准、超额收益、年化收益、Sharpe、Calmar
- 最大回撤、胜率、盈亏比
- 交易成本、滑点、换手率
- 未来函数、参数过拟合、样本内/样本外

## 模块 6：多资产与组合思维
目标：从“单标的”升级到“组合与风险预算”。
- 分散化
- 组合收益与组合风险
- 相关性与风险暴露
- 简单资产配置
- 因子组合的初步概念

## 模块 7：股票量化研究框架
目标：贴近股票量化研究的典型工作流。
- 横截面选股直觉
- 因子入门：价值、动量、质量、低波动
- 行业中性与风格暴露的基础直觉
- A股 / 美股数据结构差异
- 财务数据与价格数据的结合方式

## 模块 8：风控与执行
目标：补上“从回测到真实交易”最容易忽视的部分。
- 仓位控制
- 止损与风险预算
- 策略容量与流动性
- 执行误差
- 交易日志与复盘机制

## 模块 9：从股票扩展到 ETF / 期货 / 期权
目标：为后续扩展实操范围做准备。
- ETF 与指数策略
- 商品 / 期货趋势跟踪入门
- 黄金、白银、原油等商品与相关资产联动
- 期权与衍生品的基础认知（仅入门）

## 模块 10：个人量化实战项目
目标：把前面的知识整合成可执行项目。
- 选一个方向做完整策略研究
- 形成研究假设、数据处理、回测、评估、改进完整闭环
- 建立个人策略文档模板

## 单元目标

### 第一阶段（建议 4–6 周）
重点：模块 0–2
- 能熟练处理基础行情数据
- 能解释收益率、复利、波动率、回撤
- 能用 Python 完成基础统计与图表

### 第二阶段（建议 6–10 周）
重点：模块 3–5
- 能独立写出简单趋势/动量/均值回归策略
- 能跑基础回测并解释主要指标
- 能识别常见回测陷阱

### 第三阶段（建议 8–12 周）
重点：模块 6–8
- 能构造简单组合
- 能做基础风险控制
- 能理解实盘和回测的差异

### 第四阶段（持续进化）
重点：模块 9–10
- 能围绕一个个人关注方向做深入研究
- 例如 A股板块、ETF、贵金属、能源、期货趋势等

## 作业

### 作业设计原则
- 每课必有练习
- 作业分为三层：
  1. 概念题：确认理解
  2. Python 题：确认动手能力
  3. 研究题：培养量化思维

### 典型作业类型
- 计算收益率、波动率、回撤
- 画价格与收益率图
- 比较两只资产的相关性
- 写一个简单均线策略并回测
- 加入手续费后重新评估策略
- 比较样本内与样本外结果

---

## 里程碑

### 里程碑 1：建立量化语言（第 2–3 周）
- 已掌握基础 Python 数据处理
- 能读懂价格、收益率、波动率类图表

### 里程碑 2：会写第一个策略（第 6–8 周）
- 能写出简单信号
- 能计算策略收益和回撤

### 里程碑 3：会做第一个像样的回测（第 10–14 周）
- 能加入交易成本
- 能解释回测结果是否可信

### 里程碑 4：会做一个完整研究小项目（第 4–6 个月）
- 有自己的研究方向
- 有自己的策略文档和复盘机制

### 里程碑 5：开始小规模实操（视成熟度决定）
- 用小资金或模拟盘测试
- 强调风控优先于收益冲动

## 学员常见卡点
- 公式学过，但不会用于市场数据
- 只会看价格，不会看收益与风险
- Python 会一点，但组织不了研究代码
- 容易被“历史赚钱曲线”误导
- 容易把量化理解成预测器，而不是研究与决策框架
- 容易忽视交易成本、滑点和仓位控制
- 想太快赚钱，导致基础没有打稳
